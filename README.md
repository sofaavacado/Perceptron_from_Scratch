# Perceptron from Scratch (with Logistic Activation)

## Обзор

В этом проекте реализована модель бинарной классификации с нуля — **перцептрон с сигмоидной функцией активации (логистическая регрессия)**.

Проект охватывает полный цикл построения модели:
- генерация и нормализация данных,
- реализация функции активации и функции потерь,
- обучение с помощью градиентного спуска,
- применение L2-регуляризации,
- визуализация процесса обучения.

Цель — продемонстрировать понимание работы линейных моделей классификации и их обучения «с нуля», без использования готовых ML-библиотек.

---

## Ключевые возможности

- Реализация **перцептрона с сигмоидой (logistic perceptron)**
- Генерация синтетического датасета
- Нормализация признаков
- Численно стабильная реализация сигмоиды
- Функция потерь: **binary cross-entropy**
- Градиентный спуск
- L2-регуляризация
- Оценка accuracy
- Визуализация обучения (loss и accuracy)

---

## Что демонстрирует проект

- Понимание перцептрона и его обобщения через сигмоиду  
- Связь между перцептроном и логистической регрессией  
- Умение реализовывать ML-модели с нуля  
- Работа с градиентами и оптимизацией  
- Анализ качества модели через метрики  

---

## Архитектура проекта

### Основные этапы

1. **Генерация данных**  
   Синтетический датасет создаётся с помощью линейной модели и сигмоиды.

2. **Нормализация признаков**  
   Приведение признаков к одному масштабу для стабильного обучения.

3. **Обучение модели**  
   Параметры обновляются с помощью градиентного спуска с L2-регуляризацией.

4. **Оценка качества**  
   Accuracy рассчитывается на тестовой выборке.

5. **Визуализация**  
   Строятся графики изменения loss и accuracy.

---

## Реализованные компоненты

- **generate_data** — генерация и нормализация данных  
- **sigmoid** — функция активации  
- **cross_entropy_loss** — функция потерь  
- **train_perceptron** — обучение модели  
- **accuracy** — метрика качества  
- **plot_training_history** — визуализация обучения  

---

## Метрики и визуализация

Во время обучения отслеживаются:

- **Loss (cross-entropy)**  
- **Accuracy (%)**

После обучения:
- модель оценивается на тестовой выборке,
- строятся графики динамики обучения.
